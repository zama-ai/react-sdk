# @zama-fhe/shared

> **Internal package with shared utilities, types, and ABIs**

This package contains shared code used by both `@zama-fhe/core-sdk` and `@zama-fhe/react-sdk`. It is marked as internal and not intended for direct use by end users.

## Contents

- **[Chains](chains.md)** - Chain definitions for Sepolia and Hardhat
- **[Types](types.md)** - Shared TypeScript type definitions
- **[ABIs](abis.md)** - Contract ABIs for ERC7984 and related contracts
- **[Utilities](utilities.md)** - Validation and formatting helpers

## API Reference

See the [TypeScript API Documentation](../api/modules/_zama-fhe_shared.html) for complete reference.

---

**Note:** This is an internal package. For application development, use `@zama-fhe/core-sdk` or `@zama-fhe/react-sdk`.
